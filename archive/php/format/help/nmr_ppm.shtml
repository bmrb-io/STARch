NMRView PPM file format

Note: starting from v.4 NMRView stores data in NMR-STAR format (see The NMRView database section in the manual). You can convert individual tables from these files to the latest NMR-STAR version by selecting NMR-STAR previous version(s) as file type.

Upload file example: 

2.CA      51.484 1
2.HA       4.113 1
2.CB      19.395 1
2.QB       1.464 1
3.CA      59.673 1
3.HA       4.545 1
3.CB      69.530 1
3.HB       3.936 1
3.CG2     21.437 1
3.QG2      1.174 1

You will need to type (or paste) in the residue sequence. You may also need to provide the number of the first residue in sequence string, if the numbering in PPM file does not match the sequence (or you can edit the sequence string to match the PPM file).
