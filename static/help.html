NMR-STAR 3.1 table editor

1. Upload an NMR-STAR 3.1 data table

The file should start with loop_ and end with stop_.

Uploaded table or parser error messages (if the file was not valid) will be displayed.

Note: the table will include all NMR-STAR tags, not just those found in the input file (including Sf_ID, though this one will be omitted when printing the table out to NMR-STAR).

2. Click on a column header (tag name) to bring up the editing form.

Editing form lets you do something to the selected column, for example fill it with value or copy its values to another column.

3. Choose an editing function and value and click on update table.

You will be returned to table display, with the update showing.

You can repeat this step as many times as you like.

4. When you're done, click on Make STAR button.

Updated table will be shown as NMR-STAR loop. Use your browser's File - Save As menu to save it.

Note: when program generates NMR-STAR it deletes the uploaded table from its internal memory. You cannot use back button to return to table display and do more editing, you need to restart from the beginning and re-upload the file.
